# Social Links Profile

A clean and responsive **Social Links Profile card**, built as a Frontend Mentor challenge using **HTML5 and CSS3**.

## 🚀 Overview

## 🔗 [Project Live Link](https://social-links-profile-ebon-five.vercel.app/)

This project recreates a modern **social profile card UI** with personal information and quick access social links.

It includes:

* Profile image
* Name and location
* Short personal quote
* Social media buttons
* Hover effects
* Responsive card sizing

---

## 🛠️ Built With

* **HTML5** semantic structure
* **CSS3** custom styling
* **Flexbox**
* **Custom local font with `@font-face`**
* **Responsive media queries**

---

## 📱 Responsive Design

Optimized for:

* **Mobile** → wide centered card
* **Tablet** → smaller balanced width
* **Desktop** → compact centered profile card

Breakpoints used:

* `min-width: 600px`
* `601px - 1100px`
* `1101px+`
* `1400px+`
